# rastreioteaemadultos
Questionários de autopreenchimento podem ajudar na identificação de sinais sugestivos de TEA (Transtorno do Espectro Autista). Eles funcionam como ferramentas de triagem, mas não substituem avaliação médica especializada. O diagnóstico deve considerar história clínica, comportamento, desenvolvimento e avaliação profissional adequada.
